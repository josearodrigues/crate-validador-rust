# Código de Conduta

Este projeto adota um ambiente aberto, respeitoso e acolhedor para todos.

Esperamos que todos os participantes:

- sejam respeitosos;
- aceitem críticas construtivas;
- colaborem para melhorar o projeto;
- ajudem iniciantes.

Não serão tolerados:

- ataques pessoais;
- assédio;
- discriminação;
- linguagem ofensiva.

Caso ocorra algum problema, abra uma Issue ou entre em contato com o mantenedor.

Obrigado por contribuir para uma comunidade saudável.
